Project 14′

Recreating the Windows 7 SP1 desktop experience (2014)

What is Project 14′?

Project 14′ is a UX restoration project focused on rebuilding the Windows 7 SP1 desktop experience as it existed in 2013–2014, using modern Windows systems.

Design Principles

  • Density over touch-friendly layouts
  • Text-first interfaces
  • Blur and tint instead of transparency
  • Clear visual state communication
