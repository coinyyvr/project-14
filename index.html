What is Project 14′?

Project 14′ is a cross-platform UI restoration project focused on speed, navigation, and interaction patterns from the mid-2010s, recreated on modern operating systems without root/hacks or instability.

Platforms

  • Windows 11 (24H2) → Windows 10 behavior
  • Android 12 → Lollipop-era interaction
